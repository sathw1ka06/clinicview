\# CliniView AI - Clinical Image Analysis Platform

CliniView AI is a Flutter-based medical image analysis application that assists healthcare professionals in documenting, reviewing, and comparing clinical cases. The platform integrates AI-powered image analysis with annotation and case management features to streamline clinical workflows.

## Features

* User Authentication

  * Secure user registration and login
  * API-based authentication and user management

* AI-Powered Clinical Analysis

  * Upload clinical images for automated analysis
  * Generate detailed descriptions of visible clinical conditions
  * Receive AI-assisted diagnostic insights and estimated measurements

* Case Management

  * Store and manage previous clinical cases
  * Review historical case descriptions
  * Track case creation timestamps
  * Delete unwanted cases

* Image Annotation System

  * Create annotations on specific image regions
  * Add notes and comments to annotations
  * Edit and manage annotation data
  * Delete annotations and notes

* Case Comparison

  * Select multiple previous cases
  * Compare clinical images and findings
  * Support longitudinal case review and analysis
